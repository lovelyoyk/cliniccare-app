# ClinicCare App

Sistema completo de agendamento e gerenciamento de consultas médicas, histórico de pacientes e controle de prontuários.

## Tecnologias Utilizadas
- Java
- Spring Boot
- MySQL
- Thymeleaf

## Funcionalidades
- Agendamento online de consultas
- Gerenciamento de pacientes
- Histórico médico completo
- Relatórios para clínicas

## Status
✅ Projeto finalizado e funcional.
